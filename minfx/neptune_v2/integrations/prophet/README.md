# Prophet Integration

This module provides integration with Facebook's Prophet time series forecasting library.

## Purpose

The Prophet integration enables tracking of Prophet forecasting models, including model parameters, forecasts, and diagnostic plots.

## Installation

This integration requires the `neptune-prophet` package:
```bash
pip install neptune-prophet
```

## Functionality

The Prophet integration provides:
- Automatic logging of model parameters
- Forecast visualization
- Diagnostic plot tracking
- Model performance metrics

## Usage

The integration is automatically available when the `neptune-prophet` package is installed. Log Prophet models and forecasts through this module.
