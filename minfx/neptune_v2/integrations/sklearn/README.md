# Scikit-learn Integration

This module provides integration with the scikit-learn machine learning library.

## Purpose

The scikit-learn integration enables tracking of scikit-learn models, including parameters, metrics, and model artifacts.

## Installation

This integration requires the `neptune-sklearn` package:
```bash
pip install neptune-sklearn
```

## Functionality

The scikit-learn integration provides:
- Model parameter logging
- Cross-validation results tracking
- Feature importance visualization
- Model serialization and versioning
- Classification and regression metrics

## Usage

The integration is automatically available when the `neptune-sklearn` package is installed. Log scikit-learn models and results through this module.
