# Patches

This module provides patches and workarounds for third-party library issues.

## Purpose

The patches module contains modifications to third-party library behavior to fix bugs or adapt functionality for Neptune's specific needs.

## Key Components

- `bravado.py`: Patches for the Bravado OpenAPI client library

## Functionality

This module:
- Fixes issues in third-party libraries
- Adapts third-party library behavior for Neptune's use cases
- Provides workarounds for library limitations

## Usage

Patches are applied automatically when the Neptune client initializes, ensuring correct behavior of third-party dependencies.
